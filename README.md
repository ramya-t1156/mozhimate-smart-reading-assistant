# மொழிMate – Smart Real-Time Reading Assistant

A web-based application for real-time vocabulary learning using **Natural Language Processing (NLP)**, **Computer Vision**, and **Deep Learning**. The system helps users understand unfamiliar words from **text, speech, images, PDFs, and live video**, while providing revision, quizzes, and learning analytics.

## Technologies Used

- Python
- Flask
- HTML, CSS, JavaScript
- MongoDB
- OpenCV
- Tesseract OCR
- NLTK
- TensorFlow / Keras
