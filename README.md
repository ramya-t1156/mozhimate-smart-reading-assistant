# மொழிMate – Smart Real-Time Reading Assistant

A web-based application for real-time vocabulary learning using **Natural Language Processing (NLP)**, **Computer Vision**, and **Deep Learning**. The system helps users understand unfamiliar words from **text, speech, images and PDFs while providing revision, quizzes, and learning analytics.
